# UUSBDriver
Uniform USB System Busified Driver
通用跨平台USB驱动框架

## 解释
* **加粗**说明任务正在进行中
* [ ] 说明任务未开始
* [x] 说明任务已完成

## 项目目标
* [ ] 能正常运行
* [ ] 非线性编程模型支持(异步&多线程)
* [ ] 跨OS（Linux上能运行）
* [ ] 支持基本的usb类型(键盘/鼠标/存储/摄像头)
